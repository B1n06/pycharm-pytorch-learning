## 📝 踩坑笔记 & 常用代码备忘 (Notes)


*   **常用套路**：训练循环基本上都是 `前向传播 -> 计算 Loss -> 梯度清零(optimizer.zero_grad()) -> 反向传播(loss.backward()) -> 步进(optimizer.step())`。
*   **常见报错1**：如果在全连接层遇到 Tensor size mismatch 报错，多半是 `view()` 或者 `flatten()` 展平图片时的维度算错了。
